NovelForge - 小说设定助手
Claude 开发者手册 v1.3（完整正式版）
项目名称：NovelForge 版本：v1.3 更新日期：2026年6月 目标用户：中文小说作者（专注前期设定） 核心定位：清晰、直观、本地优先的小说世界观 + 角色 + 大纲 + 知识图谱创作辅助工具

1. 项目总览
用户核心需求：
	•	专注于小说前期创作（世界观构建、角色人物卡、大纲设计）
	•	界面必须清晰、一目了然，主要功能一秒可找到
	•	强大的项目管理功能（保存、加载、修改、导出）
	•	支持知识图谱可视化（角色关系、世界元素关联）
	•	本地优先，支持本地与云端模型混合使用
	•	适配用户硬件：16GB显存 + 32GB内存 + LM Studio

4. 功能模块详细设计
模块 1：项目管理（首页）
	•	新建项目（名称、类型、简介）
	•	项目列表（卡片视图，按修改时间排序）
	•	项目搜索
	•	打开、删除项目（带确认）
	•	一键导出项目（ZIP压缩包）
	•	项目统计（角色数量、字数等）
模块 2：世界观设定
	•	分模块结构化编辑（Accordion 或 Tabs）：
	•	基本世界信息
	•	力量/魔法/科技体系
	•	势力分布与冲突
	•	历史时间线
	•	文化、宗教、习俗、禁忌
	•	自定义模块
	•	一键生成世界观
	•	模块迭代扩展
	•	一致性检查
	•	Markdown 编辑与预览
模块 3：角色管理系统
	•	角色列表（卡片网格 + 表格切换）
	•	角色卡详细字段：
	•	基础信息、别名、年龄、身份
	•	外貌特征
	•	性格特质（MBTI + 核心欲望/恐惧）
	•	背景故事
	•	角色弧光
	•	角色关系
	•	习惯、口癖、弱点
	•	批量生成角色
	•	角色卡导出（Markdown）
模块 4：大纲设计器
	•	大纲模板选择（三幕结构、英雄之旅、雪花法等）
	•	分章编辑（章节号、标题、主要事件、冲突、高潮、关联角色、关联世界观）
	•	全局大纲概览
	•	自动关联检测
	•	一键生成大纲
模块 5：知识图谱
	•	使用 Mermaid 渲染：
	•	角色关系图
	•	世界元素关联图
	•	AI辅助生成关系描述
	•	点击节点跳转编辑页面
	•	保存为 knowledge_graph.json + Mermaid 代码
模块 6：模型与设置
	•	模型切换下拉框（Qwen / DeepSeek-v4-flash / Claude）
	•	参数调节（Temperature、Max Tokens 等）
	•	连接状态显示
	•	项目设置

5. 技术栈
	•	前端：Gradio Blocks（推荐 Soft 或 Dark 主题）
	•	LLM 接入：LiteLLM
	•	存储：每个项目一个独立文件夹（projects/项目名称/）
	•	关系图：Mermaid（浏览器端渲染）
	•	语言：Python 3.11+

6. 项目存储结构
projects/小说名称/
├── metadata.json
├── world.md
├── characters.json
├── outline.md
├── knowledge_graph.json
├── settings.json
└── exports/          # 导出文件

7. LLM Router 规范
需实现 LLMClient 类，支持：
def generate(prompt: str, model: str, temperature=0.78, max_tokens=6000) -> str
支持模型参数："qwen", "deepseek", "claude"
